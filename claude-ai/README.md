# Instagram Carousel Skill — Claude.ai Setup (No Coding Required)

By **Jeevan Bavandla** · [@jeevanbavandla](https://www.instagram.com/jeevanbavandla)

---

## What This Is

A skill file you paste into Claude.ai. Once set up, you can ask Claude to build Instagram carousels in your brand — just by describing the topic.

No terminal. No Python. No installation.

---

## What You'll Need

- A free [Claude.ai](https://claude.ai) account
- Chrome (for opening and exporting the carousel)
- 5 minutes for first-time setup

---

## Setup (One Time)

**Step 1 — Open `skill.md`**

Open the `skill.md` file from this folder. You can open it in:
- Notepad (Windows)
- TextEdit (Mac)
- Any code editor

**Step 2 — Fill in your brand details**

At the very top of `skill.md`, you'll see this section:

```
## My Brand Config

Name:           Your Full Name
Handle:         @yourhandle
Subtitle:       Your Tagline | Your Niche
Accent Color:   #6EE421
Initials:       YN
Default Style:  auto
```

Replace the placeholder values with your own:

| Field | Example |
|-------|---------|
| Name | Jeevan Bavandla |
| Handle | @jeevanbavandla |
| Subtitle | Performance Marketing \| Growth Marketing |
| Accent Color | `#6EE421` (or any hex color) |
| Initials | JB |
| Default Style | `auto` (recommended) |

**Accent color quick-pick:**
- Green → `#6EE421`
- Coral/Red → `#e8553d`
- Blue → `#3b82f6`
- Purple → `#8b5cf6`
- Orange → `#f97316`

**Step 3 — Create a Claude.ai Project**

1. Go to [claude.ai](https://claude.ai)
2. Click **Projects** in the left sidebar
3. Click **New project**
4. Name it: `Instagram Carousels`

**Step 4 — Paste the skill**

1. Inside your new project, click **Add project instructions**
2. Select all the text in `skill.md` (Ctrl+A / Cmd+A)
3. Paste it into the Project Instructions box
4. Click **Save**

That's it. Your carousel designer is ready.

---

## Creating Carousels

Start a new conversation inside your **Instagram Carousels** project and just describe what you want:

```
Create a carousel about 5 Meta Ads mistakes
```
```
Make a tweet-style carousel with my take on AI in marketing
```
```
Build an editorial carousel explaining the AIDA framework
```
```
Showcase my top 5 AI tools for marketers
```

Claude will generate the full carousel and give you a complete HTML code block.

---

## Exporting to Instagram

**Step 1 — Save the HTML**

1. Copy the entire HTML code block Claude gives you
2. Open Notepad / TextEdit
3. Paste it in
4. Save as `carousel-topic-name.html`
   - On TextEdit (Mac): Format → Make Plain Text first, then save

**Step 2 — Open in Chrome**

Double-click the `.html` file. It will open in your browser and look like an Instagram post.

**Step 3 — Export**

Click **Export All (PNG)** at the top — this downloads one PNG per slide, each at 1080×1350px (Instagram-ready).

Or click **Export as PDF** to get all slides in a single file.

**Step 4 — Upload to Instagram**

Upload the PNG files to Instagram as a carousel post.

---

## Changing Your Brand Later

To update your name, color, or anything else:

1. Go to your Claude.ai Project → **Edit Project Instructions**
2. Find the **My Brand Config** section at the very top
3. Update the values
4. Save

All future carousels will use the new settings.

---

## Tips

- **Always use Chrome** for the best export quality
- **Style auto-detection** works well — just describe what you want and Claude picks the right look
- **7 slides** is the sweet spot — not too short, not too long
- If a carousel doesn't look right, just ask Claude to adjust specific slides
